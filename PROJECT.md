# Project — Chateau Borovskoe

## Описание
Сайт закрытой продажи дома (~2,7 млрд ₽ / $33M) в стиле французского абсолютизма.
Архитектор Моро. Боровское шоссе, 14 км от МКАД.

## Секции сайта
1. **Hero** — полноэкранное фото фасада, ключевые цифры, CTA
2. **Объект** — описание + таблица фактов (площадь, участок, спальни, цена)
3. **Особняк** — галерея фасадов (7 фото)
4. **Интерьеры** — галерея интерьеров (7 фото) + список особенностей (8 пунктов)
5. **Территория** — фото парка + спецификации автономии (5 пунктов)
6. **Расположение** — таблица расстояний (5 направлений)
7. **Атмосфера** — полноэкранное фото статуй
8. **Заявка** — текст (форма временно отключена)

## Декоративные элементы
- Колонны (column1-4.png) — по бокам секций, в полный рост
- Розы (roses.png) — из углов секций
- Облака (clouds.png) — фоновые акценты
- Бюст (bust.png) — секция Расположение
- Херувим (cherub.png) — не используется сейчас
- SVG-лоза — секция Интерьеры

## Функциональность
- Lightbox для фотографий (клик → полноэкран → листание → закрытие)
- Двуязычность RU/EN с переключателем
- Активная секция подсвечивается в навигации
- Параллакс декоративных элементов при скролле
- Scroll-reveal анимации
- Адаптив: desktop (>900px) и mobile (<900px) с раздельным декором
