# Project — Premium House Sale

## Описание
Сайт закрытой продажи дома (~2,7 млрд ₽ / $33M) в стиле французского абсолютизма.
Архитектор Моро. Боровское шоссе, 14 км от МКАД.

## Статус: ГОТОВ
- Десктопная версия — утверждена
- Мобильная версия — утверждена
- Lightbox фотогалерея — работает
- Деплой на Vercel — активен

## Секции сайта
1. **Hero** — полноэкранное фото фасада, цена, площадь, участок, спальни, CTA
2. **Объект** — описание + таблица фактов + колонна справа
3. **Особняк** — галерея фасадов (7 фото) + колонна с цветами слева + розы справа
4. **Интерьеры** — галерея (7 фото) + 8 особенностей + SVG-лоза из углов
5. **Территория** — фото парка + 5 спецификаций автономии + колонна + розы
6. **Расположение** — 5 направлений + облака + бюст
7. **Атмосфера** — полноэкранное фото статуй (разделитель)
8. **Заявка** — текст + SVG-лоза (форма отключена, готова к подключению)

## Функциональность
- **Lightbox** — клик на фото → полноэкран, стрелки ‹›, Escape, счётчик
- **Двуязычность** — RU/EN переключатель
- **Навигация** — золотая подсветка активной секции
- **Параллакс** — декоративные элементы при скролле
- **Scroll-reveal** — контент появляется при прокрутке
- **Адаптив** — раздельный декор desktop/mobile (`desktop-only`/`mobile-only`)
