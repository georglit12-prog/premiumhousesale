# Project — Premium House Sale

## Описание
Сайт закрытой продажи дома (~2,7 млрд ₽ / $33M) в стиле французского абсолютизма.
Архитектор Моро. Боровское шоссе, 14 км от МКАД.

## Секции сайта
1. **Hero** — полноэкранное фото фасада, цена, площадь, участок, спальни, CTA
2. **Объект** — описание + таблица фактов с декоративной колонной
3. **Особняк** — галерея фасадов (7 фото) с колонной и розами
4. **Интерьеры** — галерея (7 фото) + 8 особенностей + SVG-лоза
5. **Территория** — фото парка + 5 спецификаций автономии + колонна и розы
6. **Расположение** — 5 направлений + облака и бюст
7. **Атмосфера** — полноэкранное фото статуй (разделитель)
8. **Заявка** — текст «Будем рады организовать просмотр» + облако (форма отключена)

## Функциональность
- **Lightbox** — клик на фото → полноэкран, стрелки/свайп, Escape для закрытия
- **Двуязычность** — RU/EN переключатель в навигации
- **Навигация** — золотая подсветка активной секции при скролле
- **Параллакс** — декоративные элементы сдвигаются при скролле
- **Scroll-reveal** — контент появляется при прокрутке
- **Адаптив** — раздельный декор для desktop (>900px) и mobile (<900px)
