# Дом по авторскому проекту · Боровское шоссе

Сайт-презентация закрытой продажи. Двуязычный (RU/EN), заявки приходят в Телеграм.

## Что внутри

| Файл | Назначение |
|---|---|
| `index.html` | Сайт (лёгкая версия, использует папку `assets/`). Это файл для публикации. |
| `assets/` | Фотографии: реальные снимки объекта + атмосферные кадры стиля. |
| `api/submit.js` | Серверная функция: принимает заявку и шлёт её в Телеграм. **Токен хранится в настройках хостинга, в коде его нет.** |
| `presentation-standalone.html` | Та же страница, но **одним файлом** (фото внутри). Удобно переслать другому агенту или открыть без интернета. *Форма в этом режиме работает только если прописать полный адрес — см. ниже.* |
| `package.json` | Технический файл для Vercel. |

---

## Шаг 1. Телеграм (3 минуты)

1. Откройте **@BotFather** → `/newbot` → получите **токен** (выглядит как `1234:AAA...`).
   - Если бот уже создан, но токен мог «засветиться» — пришлите `/revoke` и получите новый.
2. Откройте **@userinfobot** → он пришлёт ваш **chat_id** (число).
3. **Напишите своему боту любое сообщение** (например «старт»). Без этого Телеграм не даст боту прислать вам заявку.

---

## Шаг 2. Публикация на Vercel (бесплатно)

1. Зайдите на **vercel.com**, войдите (можно через GitHub/почту).
2. Нажмите **Add New → Project**.
3. Загрузите эту папку (перетащите её целиком) либо подключите репозиторий.
4. До нажатия Deploy откройте **Environment Variables** и добавьте две переменные:

   | Name | Value |
   |---|---|
   | `TELEGRAM_BOT_TOKEN` | ваш токен от BotFather |
   | `TELEGRAM_CHAT_ID` | ваш chat_id |

5. Нажмите **Deploy**. Через минуту получите ссылку вида `https://ваш-проект.vercel.app` — это и есть готовый сайт. Эту ссылку и пересылайте.

После заполнения формы заявка (ФИО агента, ФИО покупателя, телефон) придёт вам в Телеграм.

## Цена

Цена задаётся в одном месте — в самом низу `index.html`, блок `var PRICE`:

```js
var PRICE = {
  ru: { short: '2,7 млрд ₽', full: '2 700 000 000 ₽' },
  en: { short: '$33M',       full: '$33,000,000' }
};
```

> ⚠️ В присланных PDF указано **$35 млн**, а в задании — **$33 млн / 2,7 млрд ₽**. Сейчас стоит **$33M / 2,7 млрд ₽**. Если верная цена другая — поменяйте обе строки.

## Про отдельный файл (`presentation-standalone.html`)

Это вся презентация в одном файле — открывается двойным кликом где угодно, удобно пересылать. Но форма заявки в нём по умолчанию ищет сервер на том же адресе. Чтобы заявки уходили и из пересланного файла, найдите в нём строку:

```js
var FORM_ENDPOINT = '/api/submit';
```

и впишите полный адрес вашего развёрнутого сайта:

```js
var FORM_ENDPOINT = 'https://ваш-проект.vercel.app/api/submit';
```

## Важно про фотографии

Атмосферные кадры (закаты, витражи, скульптуры, лепнина) — это **визуализация стиля**, они помечены подписью «Визуализация стиля · не фотография объекта». Реальные фотографии особняка находятся в разделах **«Особняк»**, **«Интерьеры»** и **«Территория»** — они показаны чисто, без художественных наложений.
