# Premium House Sale — Боровское шоссе

Сайт-презентация закрытой продажи дома по авторскому проекту в стиле французского абсолютизма.

**Live:** [premiumhousesale.vercel.app](https://premiumhousesale.vercel.app)

## Что внутри

| Файл | Назначение |
|---|---|
| `index.html` | Основной сайт — адаптивный, двуязычный (RU/EN) |
| `assets/` | Фотографии объекта (фасады, интерьеры, территория) |
| `assets/decor/` | Декоративные элементы без фона (колонны, розы, облака, бюст) |
| `api/submit.js` | Серверная функция для отправки заявок в Telegram (не подключена) |

## Быстрый старт

Сайт опубликован на Vercel. Для подключения формы заявок:

1. Создайте бота через **@BotFather** в Telegram → получите токен
2. Узнайте свой **chat_id** через **@userinfobot**
3. В настройках проекта на **vercel.com** добавьте Environment Variables:
   - `TELEGRAM_BOT_TOKEN` — токен бота
   - `TELEGRAM_CHAT_ID` — ваш chat_id
4. Передеплойте проект

## Цена

Задаётся в `index.html`, блок `var PRICE`:

```js
var PRICE = {
  ru: { short: '2,7 млрд ₽', full: '2 700 000 000 ₽' },
  en: { short: '$33M',       full: '$33,000,000' }
};
```
